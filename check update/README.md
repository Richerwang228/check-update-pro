# 视频更新监控与管理 GUI 工具

这是一个基于Python的桌面应用程序，用于自动检查视频创作者的更新情况，并提供清晰的图形界面展示。

## 主要功能

- 自动监控UP主视频更新
- 支持添加、删除、修改UP主书签
- 智能判断新视频更新
- 支持标记视频为"已看"
- 可配置的检查周期和更新范围
- 美观的图形用户界面

## 安装要求

- Python 3.8+
- 见 requirements.txt 文件获取完整依赖列表

## 安装步骤

1. 克隆或下载本项目
2. 安装依赖：
```bash
pip install -r requirements.txt
```

## 使用方法

运行主程序：
```bash
python main.py
```

## 配置说明

- 在设置界面可以配置：
  - 检测周期
  - 更新判定的时间范围
  - 自动检测开关
  - 其他偏好设置

## 注意事项

- 请合理设置检测频率，避免过于频繁的请求
- 建议使用代理以防止IP被封禁
- 定期备份书签数据
